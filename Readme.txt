To run the simulation of part 1:
Make sure to be in the same directory as the simulation file and run the following command in the terminal:
    python simulation.py
This will run the simulation and generate a txt file with the results.

To run the simulation of part 2 and 3:
Import the .unitypackage file into Unity
Make sure to be in the same directory as the simulation file and run the both server.py files in their corresponding directories:
    python server.py
Then start the scene in Unity
